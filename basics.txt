Install playwright via npm or another source from https://playwright.dev/docs/intro 


Confirm additional browser installation when prompted during install

Install dotenv:
npm install dotenv

Pull https://github.com/stefanpejcic/openpanel-tests  and

Create a new .env file inside openpanel/.env or openadmin/.env to setup the authentication params:

BASE_URL=https://185.193.66.252:2083
PANEL_USERNAME=yourusername
PANEL_PASSWORD=yourpassword



To access playwright UI cd into openpanel-tests/ (project root) and run 

npx playwright test -c openpanel/playwright.config.ts --project=tests --project=tests --ui



This is where you run existing tests by clicking the play button under tests


To create a new test script simply give it a .config.ts extension and place it in openpanel/tests/ or openadmin/tests/ , e.g. filemanager.spec.ts

You can use codegen to generate test scripts by doing actions manually in playwright browser

Example command to start playwright codegen:

npx playwright codegen https://OPENPANEL_URI


Useful commands: https://github.com/stefanpejcic/openpanel-tests/blob/main/README.md
